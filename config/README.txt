# Letakkan file JSON Service Account Google di sini
# dengan nama: service-account.json
#
# Cara mendapatkannya:
# 1. Buka https://console.cloud.google.com/
# 2. IAM & Admin → Service Accounts → Create Service Account
# 3. Buat key bertipe JSON
# 4. Download dan simpan sebagai config/service-account.json
#
